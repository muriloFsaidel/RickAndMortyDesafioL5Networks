#Desafio L5 Networks Angular Rick And Morty API

  <p>O projeto foi concebido a partir de um desafio proposto pela empresa L5 para a vaga de desenvolvedor Front-end Angular Júnior. 
   O desafio se resume em construir um projeto fron-end do zero, apenas com as orientações do que precisa ser feito, e este projeto tem o objetivo
   de buscar dados da API pública chamada Rick And Morty API(https://rickandmortyapi.com/api), e apresentá-los através de tabelas dinâmicas e navegação por rotas.
   </p>
   
#Instalação
  
  ° Possuir Node.js instalado no mínimo versão 22.11.0 (versão da minha máquina); <br>
  ° No terminal, acesse a raiz do projeto, e execute npm install para instalar as dependências;<br>
  º Se não estiver com o ngCLI instalado, instale via npm install -g @angular/cli;<br>
  ° Para subir a aplicação, utlize o ng serve.<br>
